📄 1. Introduction

This Privacy Policy describes how GEE Pal AI, a Chrome Extension developed by notGIS, handles your information. We are committed to protecting your privacy and being transparent about our data practices.

GEE Pal AI is an AI-powered assistant for Google Earth Engine (GEE) that helps users generate, modify, and fix GEE JavaScript code using third-party AI APIs.

✅ Short version: We do not collect, store, or transmit any personal data to our own servers. Your API keys are stored only on your local device.

🗂️ 2. Data We Collect

GEE Pal AI collects no personal data on our own servers. However, the extension stores the following data locally on your device only using Chrome's built-in chrome.storage.local API:

  • Your API keys (Routeway API key and/or Google Gemini API key)
  • Your selected AI model preferences
  • Your last used AI assistant session URL
🔒 All data is stored locally on your device. It is never transmitted to notGIS servers or any third party other than the AI API provider you choose to use.

🔌 3. Third-Party API Services

This extension enables you to connect to third-party AI APIs using your own API keys. You are solely responsible for your use of these services and must comply with their respective Terms of Service and Privacy Policies.

Service Purpose Data Sent Their Privacy Policy

If privacy is a concern, we recommend using the paid Gemini API tier, which does not use your data for model training.

🛡️ 5. Chrome Permissions Explained

GEE Pal AI requests the following Chrome permissions:

  • activeTab — To read and inject code into the active Google Earth Engine Code Editor tab. Used only on GEE pages.
  • scripting — To execute scripts in the GEE Code Editor for code injection, error detection, and button clicking.
  • storage — To save your API keys and settings locally on your device using Chrome's secure local storage.
  • tabs — To detect the active GEE tab and open AI assistant tabs when requested by the user.
✅ The extension only activates on code.earthengine.google.com and code.earthengine.google.co.in. It does not run on any other websites.

🔐 6. Data Security

Your API keys are stored using Chrome's chrome.storage.local API, which is sandboxed to this extension only. No other extension or website can access your stored API keys.

All communication with third-party AI APIs is performed directly from your browser over HTTPS (encrypted connections). notGIS does not act as a proxy and never sees your API requests or responses.

  • API keys are never transmitted to notGIS servers
  • No login or account creation required
  • No cookies are set by this extension
  • No tracking scripts or analytics are included

👤 7. User Responsibilities

By using GEE Pal AI, you agree to:

  • Provide your own valid API keys for Routeway or Google Gemini
  • Comply with the Terms of Service of each AI API provider you use
  • Comply with Google Earth Engine's Terms of Service
  • Take responsibility for the content of code prompts you submit to AI APIs
  • Secure your own API keys and not share them with others
⚠️ API Costs: Using third-party AI APIs may incur costs based on your usage. notGIS is not responsible for any charges from Routeway, Google Gemini, or any other API provider.

👧 8. Children's Privacy
